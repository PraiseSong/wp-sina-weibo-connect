== Description ==

使用新浪微博瓣账号登陆你的 WordPress 博客，并且留言使用新浪微博的头像。博主可以同步日志到新浪微博，用户可以同步留言到新浪微博。

详细介绍： http://fairyfish.net/project/sina-connect/

== Installation ==

上传激活即可。详细安装请点击： http://fairyfish.net/project/sina-connect/
